## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Set up environment
cp .env.example .env.local
# Add your API keys to .env.local

# Run development
npm run dev

# Check TypeScript
npm run typecheck

# Build for production
npm run build
```
