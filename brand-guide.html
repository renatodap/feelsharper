The Lightning Principle

The lightning bolt represents sudden transformation—the moment when potential becomes kinetic. It's nature's most dramatic display of power transfer, mirroring how Sharpened transforms human potential into realized achievement. The bolt forms an S, not just for Sharpened, but for the Sudden, Strike-like nature of breakthrough moments in personal evolution.

Logo Usage Guidelines

Do

  • Maintain clear space equal to half the logo height
  • Use official color gradients for product logos
  • Apply subtle animation on hover (scale or glow)
  • Ensure high contrast with background
  • Use vector formats for scalability

Don't

  • Rotate or skew the lightning bolt
  • Add drop shadows or effects
  • Change the bolt's proportions
  • Use low contrast combinations
  • Place over busy backgrounds

Motion & Interaction

Purposeful animation that respects user time

Animation Principles

All animations use ease-out timing for snappy, responsive feel. Duration: 200-300ms for micro-interactions, 400-600ms for page transitions. 

/* CSS Animation Standards */
.lightning-hover {
    transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}

.lightning-hover:hover {
    transform: scale(1.05);
    filter: drop-shadow(0 0 20px currentColor);
}

/* Glass Morphism Effect */
.glass-panel {
    background: rgba(255, 255, 255, 0.1);
    backdrop-filter: blur(10px);
    border: 1px solid rgba(255, 255, 255, 0.1);

// React Component Example
import { Lightning } from '@sharpened/icons';

const ProductLogo = ({ product, size = 40 }) => {
  const gradients = {
    feel: ['#4169E1', '#00BFFF'],
    study: ['#8B4FC3', '#B794F4'],
    work: ['#FF6B35', '#FFA500'],
    mind: ['#FF1493', '#FF69B4'],
    code: ['#00FF41', '#39FF14']
  };
  
  return (
    <div 
      className="logo-container"
      style={{
        background: `linear-gradient(135deg, 
          ${gradients[product][0]}, 
          ${gradients[product][1]}
        )`,
        borderRadius: 'var(--radius-lg)'
      }}
    >
      <Lightning size={size} color="white" />
    </div>
  );
};
